NAME

Lintian::Output::Universal -- Facilities for printing universal hints

SYNOPSIS

 use Lintian::Output::Universal;

DESCRIPTION

A class for printing hints using the 'universal' format.

INSTANCE METHODS

issue_hints

Print all hints passed in array. A separate arguments with processables is necessary to report in case no hints were found.
